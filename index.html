  • 15+ RESTful API endpoints supporting mobile and web clients
  • Real-time notifications using Firebase Cloud Messaging
  • Audit logs for status changes, messages, and notes
  • Complaint lifecycle management with validation on status transitions
  • Optimized queries and caching for high concurrency
  • Security: JWT Authentication, role-based access, and data validation

This system was developed for a real beauty center to replace manual booking and invoicing processes. It handles daily operations including appointment scheduling, invoice generation, employee management, and client notifications with a scalable backend architecture.

  • Role-Based Access Control (Admin, Receptionist, Client) using Spatie Permissions
  • Asynchronous background jobs (Queues) for email verification, notifications, and heavy tasks
  • Automated booking lifecycle with validation and cancellation rules
  • Invoice generation and archiving system using Jobs & Commands
  • Firebase Cloud Messaging integration for real-time client notifications
  • Clean Architecture with Service & Repository patterns
  • Optimized database queries and indexes for daily operational usage
